A test plugin for BrailleSC/Making DH More Open, experimenting with plugin presentation. Takes the text marked by shortcodes and applies a styling function.

== Description ==

A plugin that takes the text marked by shortcodes and applies a styling function.

== Installation ==

1. Upload the "reprint" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin convert post text to Braille? =

No.
